# Mouse Random Bot

這個程式會隨機移動滑鼠到螢幕上的不同位置。

## 需求

請確保已安裝以下 Python 套件：

- pyautogui
- pymouse
- pywin32 (如果在 Windows 上運行)

您可以使用以下命令安裝所有依賴：

```bash
pip install -r requirements.txt
```

## 使用方法

1. 進入 `tools` 目錄：

```bash
cd '.\Python Sample\tools\'
```

2. 運行 `mouse_random_bot.py`：

```bash
python mouse_random_bot.py
```

3. 程式會每秒隨機移動滑鼠到螢幕上的不同位置。

4. 按下 `Ctrl+C` 可以停止程式，並顯示 "stop" 訊息。
